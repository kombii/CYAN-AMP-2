# CYAN-AMP-2
1.采用TDA7498E的一台即插即用功率放大器（音频后级）；
2.设计了可供FDM模式3D打印的外壳；
3.采用远超散热需求的散热片，无需风扇，对环境要求低；
4.纯后级设计，本着放手不管的原则，不含电源开关，不含音量旋钮，可控增益开关也在外壳安装后不可触碰；
5.仅保留3.5mm音频接口输入，预留了平衡输入接口（非标设计，需要自制转接线）；
6.保留工作模式跳线，可选外部时钟输入和多机联合运行。
7.自动关机设计，未插入输入线缆时进入低功耗运行。
1. A plug-and-play power amplifier using TDA7498E (audio post-stage);
2. Designed the shell for 3D printing in FDM mode;
3. A heat sink far exceeding the heat dissipation requirement, no fan is required, and the environmental requirements are low;
4. Pure back-end design, in the principle of “let it go”, no power switch, no volume knob, and controllable gain switch are also not touchable after the shell is installed
5. Only the 3.5mm audio interface input is reserved, and the balanced input interface is reserved (non-standard design, self-made adapter cable is required);
6. Retain the working mode jumper, optional external clock input and multi-machine joint operation.
7. Automatic shutdown design, enter low-power operation when no input cable is inserted.
